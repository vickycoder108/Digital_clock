# Digital Clock

A simple and elegant digital clock web application that displays the current time and date.

## Features

- **Real-time Clock Display**: Shows hours, minutes, and seconds
- **AM/PM Indicator**: Displays whether it's morning or evening
- **Current Date**: Shows the full date with day of the week
- **Responsive Design**: Works seamlessly on different screen sizes

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Clock functionality and time updates

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The clock will automatically display the current time and update every second

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
